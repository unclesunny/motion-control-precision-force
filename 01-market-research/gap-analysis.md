# 差距分析 — vs 标杆 (PANATERM / Musashi / TwinCAT)

## 一、方法论

对 8 个维度进行 L0-L5 能力评分：
- **L0**: 不存在
- **L1**: 概念验证
- **L2**: 基本可用（单场景）
- **L3**: 生产可用（多场景）
- **L4**: 行业领先
- **L5**: 标杆/超标杆

评分来源：对标竞品分析的公开资料 + 本项目的当前交付物审查。

---

## 二、标杆选择

| 标杆 | 领域 | 理由 |
|------|------|------|
| **PANATERM** (松下) | 伺服调试软件 | 行业最成熟的伺服调试工具，4 通道示波器 + 自动调谐 |
| **Musashi ELISA** | 力控精度 | 点胶力控全球标杆，±1% 重复精度 |
| **TwinCAT Scope** (倍福) | 示波器 | 工业界最强示波器，无限通道，可编程触发 |

## 三、逐维度差距矩阵

### 3.1 示波器通道数

| 系统 | 能力等级 | 通道数 | 说明 |
|------|---------|--------|------|
| 台达 ASDA-Soft | L2 | 2 | 基本够用，无法同时观测位置+速度+电流+力矩 |
| 松下 PANATERM | L4 | 4 | 行业标配，4 通道覆盖基本调试 |
| 倍福 TwinCAT Scope | L5 | 无限 | 软件定义，无硬件限制 |
| **本项目 (当前)** | L4 | 8 | 4× 台达，2× 松下 |
| **本项目 (目标)** | L5 | 无限 | 软件通道，不受限 |

**差距**: L4 → L5 (6 个月)。当前 8 通道已超越所有伺服原厂工具，下一步是实现软件定义通道数。

### 3.2 采样率

| 系统 | 能力等级 | 最高采样率 | 说明 |
|------|---------|----------|------|
| 台达 ASDA-Soft | L1 | 500 Hz | 严重不足，高频振动无法捕获 |
| 松下 PANATERM | L3 | 1 kHz | 伺服调试够用 |
| 倍福 TwinCAT Scope | L5 | 可配 (最高 50 kHz) | 取决于硬件 |
| **本项目 (当前)** | L4 | 10 kHz (设计) | 1 kHz demo 已验证，10 kHz 待 EtherCAT DC 同步 |
| **本项目 (目标)** | L5 | 可配 (最高 50 kHz) | 取决于 EtherCAT 周期 |

**差距**: L4 → L5 (12 个月)。10 kHz 已 10× 台达、10× 松下。进一步提升需要 PREEMPT_RT 内核。

### 3.3 AI 分析

| 系统 | 能力等级 | AI 功能 | 说明 |
|------|---------|---------|------|
| 台达 ASDA-Soft | L0 | 无 | 纯人工分析 |
| 松下 PANATERM | L1 | 自动调谐 (黑盒) | 无解释、无标注 |
| 倍福 TwinCAT Scope | L0 | 无 | 纯数据采集 |
| **本项目 (当前)** | L3 | 电流异常 + 跟随误差 + FFT 谐振 | 三检测器管线已验证 (43 测试) |
| **本项目 (目标)** | L5 | PPO 自整定 + 参数推荐 + 自动诊断 | AI&ML Agent 能力接入 |

**差距**: L3 → L5 (12 个月)。**这是本项目最显著的差异化维度——所有竞品的 AI 分析都是 L0-L1。**

### 3.4 力控精度

| 系统 | 能力等级 | 精度 | 说明 |
|------|---------|------|------|
| Musashi ELISA | L5 | ±1% | 专利容积计量 + 压力反馈 |
| Nordson Performus | L5 | ±1% | 精密阀门 + 力传感器 |
| 台达 A3 (无外部力传感器) | L1 | ±5-10% (电流估算) | 无直接力控 |
| **本项目 (当前)** | L1 | 无闭环力控 | CODESYS FB 已编写，未实测 |
| **本项目 (目标)** | L3 | ±3% (PPO 闭环) | 力传感器 + EtherCAT + PPO |

**差距**: L1 → L3 (12 个月)。这是最大的硬件差距——需要力传感器 + 标定。

| 风险 | 概率 | 影响 | 缓解措施 |
|------|------|------|---------|

| 力传感器选型/Sourcing | M | H | 优先用模拟量传感器 (0-10V) 降低门槛 |
| 中日文参数翻译质量 | L | L | 已通过 CHM 反编译获取台达中文手册 |
| 竞争对手推出免费 AI 工具 | L | M | 开源生态壁垒 + 多品牌兼容是护城河 |

## 六、关键结论

1. **AI 分析是我们最大的差异化** — 所有竞品在此维度都是 L0-L1
2. **硬件差距 (力控精度) 最大** — 需要力传感器和实测，但软件侧已就绪
3. **价格和开放性已是 L5** — 保持开源是关键竞争壁垒
4. **PANATERM 是软件体验标杆** — 4 通道 1 kHz 示波器 + 自动调谐，我们 8 通道 10 kHz + AI 已在规格上超越
5. **TwinCAT 是开放性标杆** — 我们用开源替代收费授权，用中文原生超越英文界面
